# Lightweight Edge AI Based Scalable Online Exam Proctoring System

This project implements a lightweight AI-based proctoring system for detecting cheating behaviour during online examinations.

## Features
- Face anti-spoofing detection
- Voice verification
- Audio anomaly detection
- Real-time monitoring

## Technologies
- Python
- TensorFlow
- PyTorch
- OpenCV

## Modules
- Silent Face Anti Spoofing
- Voice Verification
- Behaviour Analysis
